## Mindset

사용자 경험(UX)과 사용자 인터페이스(UI)를 최우선으로 고려하며 개발합니다.

어떤 프로덕트를 만들더라도, 사용자가 직관적이고 편리하게 서비스를 이용할 수 있도록 고민합니다.

새로운 기술에 대한 배움에 열려 있으며, 다룰 수 있는 기술의 폭을 넓혀가는 동시에,

탄탄한 기본기를 바탕으로 깊이 있는 이해를 추구합니다.
